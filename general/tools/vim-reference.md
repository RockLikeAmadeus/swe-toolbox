# Modes

Normal Mode (`Esc`)
Insert Mode (`i`, `a`, `o`)
Replace Mode (`r`, `R`)
Visual Mode (`v`, `V`, `Ctrl+v`)
Command Mode (`:`)

# Critical Combinations

## Exiting Vim

Discard changes and exit: `:q!`

# Normal Mode

## Cursor Movement

Left: `h`

Right:  `l`

Up: `k`

Down: `j`

## Commands (might not be the right term)

Delete character under cursor: `x`
