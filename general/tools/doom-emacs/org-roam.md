# Org Roam Notes

# Frequent Keybinds

`SPC n r n` - (org-roam-capture) Creates a node if it doesn't exist, and restores the previous window state on completion.
`SPC n r f` - (org-roam-node-find) Node roam find - Create or look up a Roam "node"
`SPC n r i` - (org-roam-node-insert) Insert a link to another node. If the note doesn't exist it will create it.
`SPC m I` - (org-id-get-create) - Use this to make a headline within an org file into a Roam node.
`SPC n r r` - Open Roam sidebar (from within a roam note) (this is the Org-roam buffer) - View other notes that link to this note
`SPC n r s` - (org-roam-db-sync) Note roam database sync - Force re-index of files (this is wrong)

## Dailies

`SPC n r d t` - Note roam daily today - Open or create a note for the current calendar day
`SPC n r d y` - Jump to yesterday's daily note
